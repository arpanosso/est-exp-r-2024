Teste de McNemars

Utilizado para comparar propores em estudos pareados ou dependentes.

til em situaes onde os dados so categricos e se deseja verificar se h uma mudana significativa entre dois momentos ou condies, por exemplo, antes e depois de uma interveno em um mesmo grupo.

Quando usar o teste de McNemar

  • Estudos pareados ou dependentes: Como no caso de pacientes avaliados antes e depois de um tratamento, ou amostras relacionadas.

  • Variveis categricas binrias: Por exemplo, sucesso/fracasso, sim/no, presente/ausente.

  • Objetivo: Verificar se h uma diferena significativa na distribuio das categorias entre os dois momentos.

Hipteses do teste

Hiptese nula (H0): No h diferena significativa entre as propores (b = c).

Hiptese alternativa (H1): H diferena significativa entre as propores (b c).

Estatstica do teste

\[\chi^2 = \frac{(b-c)^2}{b+c} \sim \mathcal{\chi}^2_{1}\] Essa estatstica segue uma distribuio qui-quadrado com 1 grau de liberdade.

O valor crtico do teste ao alpha de 5% \((\alpha=5\%)\) \(3,84\)

# Calculando o valor tabelado
qchisq(1-0.05,1)

Anlise de correlao Phi

O coeficiente phi uma medida de associao entre duas variveis categricas binrias, ou seja, variveis que possuem exatamente duas categorias (como sim e no ou presente e ausente). Ele amplamente utilizado em anlise de correlao para variveis dicotmicas, permitindo avaliar a fora e a direo da relao entre elas.

O valor do coeficiente phi varia entre 1 e 1:

=1: Associao perfeita positiva (as duas variveis esto perfeitamente relacionadas de forma direta).
=1: Associao perfeita negativa (as duas variveis esto perfeitamente relacionadas de forma inversa).
=0: Nenhuma associao (as variveis so independentes).

library(psych)
data_set |> 
  select(blood_q_pcr, culture_q_pcr) |> 
  mutate(
    a = as.numeric(blood_q_pcr == TRUE & culture_q_pcr == TRUE),
    b = as.numeric(blood_q_pcr == TRUE & culture_q_pcr == FALSE),
    c = as.numeric(blood_q_pcr == FALSE & culture_q_pcr == TRUE),
    d = as.numeric(blood_q_pcr == FALSE & culture_q_pcr == FALSE)
  ) |> 
  summarise(
    a=sum(a),
    b=sum(b),
    c=sum(c),
    d=sum(d)
  ) |> as.matrix() |> as.vector() |> phi()
## [1] 0.1
data_set_dic <- data_set |> select(blood_q_pcr:ifa3) |> data.frame()
mat_phi <- matrix()
n <- length(data_set_dic)
mat_phi <- matrix(ncol=n,nrow = n)
for(i in 1:length(data_set_dic)){
  for(j in 1:length(data_set_dic)){
    vl1 <- data_set_dic[,i]
    vl2 <- data_set_dic[,j]
    a = sum(vl1==TRUE & vl2==TRUE)
    b = sum(vl1==TRUE & vl2==FALSE)
    c = sum(vl1==FALSE & vl2==TRUE)
    d = sum(vl1==FALSE & vl2==FALSE)
    mat_phi[i,j] <- phi(c(a,b,c,d)) 
  }
}
colnames(mat_phi) <- rownames(mat_phi) <- names(data_set_dic)
